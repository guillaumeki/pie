# Conventional Commits (Optional)
Date: 2026-02-03

Use when helpful; not mandatory unless requested.

## Format
<type>(<scope>): <description>

## Types
- feat: new feature
- fix: bug fix
- docs: documentation
- refactor: code change without behavior change
- test: tests only
- chore: tooling/maintenance

## Examples
- feat(parser): support comparison operators
- fix(session): handle UnionQuery in source detection
- docs(process): add DoD and versioning
