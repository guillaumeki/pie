# Definition of Done
Date: 2026-02-03

## Required
- Plan approved and saved in `plans/` (when the change is important).
- Code changes completed and aligned with the approved plan.
- All relevant tests executed and passing.
- `CHANGELOG.md` updated with the date (and version if changed).
- Design decisions recorded in `design/<feature>.md` when applicable.
- `AGENTS.md` updated with references to new plan/design docs.

## Recommended
- Review checklist completed.
- Release checklist updated (for release changes).
- Documentation updated (README/AGENTS) if user-facing behavior changed.
