# Usage

## Installation
```bash
pip install -e .
```

## Common Workflows
- Parse DLGPE/DLGP files with the corresponding parser.
- Build a fact base from parsed facts.
- Evaluate queries with `GenericFOQueryEvaluator` or via `ReasoningSession`.

See `README.md` for runnable examples.

## CLI
```bash
disjunctive-rewriter [file.dlgp] [-l LIMIT] [-v] [-m]
```
